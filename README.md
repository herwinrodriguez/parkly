# Parkly

Aplicación de ejemplo creada con [Next.js 14](https://nextjs.org/) y TypeScript.
Incluye integración con [Tailwind CSS](https://tailwindcss.com/) y
[shadcn/ui](https://ui.shadcn.com/) para componentes reutilizables.

## Scripts

- `npm run dev` – inicia el servidor de desarrollo
- `npm run build` – genera la versión de producción
- `npm start` – ejecuta la aplicación compilada
- `npm run lint` – ejecuta ESLint
- `npm test` – ejecuta pruebas (placeholder)
